# Clarity Analyzer

JavaFX-Application to interactively visualize the raw data of a Dota 2 replay.

![Clarity Analyzer](/screenshot.png?raw=true)

# Building

Attention: Only builds and runs with Java 8 atm!

`mvn package`

# Running

`java -jar target\analyzer.one-jar.jar`
